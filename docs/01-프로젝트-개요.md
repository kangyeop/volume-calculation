# 프로젝트 개요

## 이 프로젝트가 하는 일

물류 창고에서 출고 시 상품을 어떤 박스에 어떻게 담을지 계산해주는 dnut VC(Volume Calculator) 웹 애플리케이션입니다.

핵심 기능:
- 상품 그룹별 상품 마스터 데이터 관리
- 출고 엑셀 업로드 시 AI 기반 컬럼 자동 매핑
- 등록된 박스 규격 기준 패킹 최적화 계산
- 패킹 결과 엑셀 내보내기

## 사용자 흐름

### 1. 상품 등록

- `/products`에서 상품 그룹 생성
- `/products/:id`에서 상품 엑셀 업로드
- AI가 SKU, 상품명, 치수 컬럼을 자동 추정
- 확인 후 상품 일괄 저장

### 2. 박스 등록

- `/boxes`에서 사용 가능한 박스 규격 등록
- 박스는 전역(모든 배치에서 공유)

### 3. 출고 업로드

- `/outbound`에서 출고 엑셀 업로드
- AI가 주문번호, SKU, 수량, 수령인, 주소 컬럼을 자동 추정
- 상품명/SKU 기준으로 등록된 상품과 자동 매칭
- OutboundBatch가 생성되고 주문/출고 아이템이 저장됨

### 4. 패킹 계산

- `/outbound/:id/packing`에서 그룹핑 옵션 선택 (주문별 / 수령인별 / 주문+수령인별)
- 패킹 계산 실행
- 각 그룹별 박스 배정 결과 확인
- 엑셀로 내보내기

## 현재 구현된 범위

있는 것:
- 상품 그룹/상품 CRUD
- 출고 배치/주문/아이템 관리
- AI 기반 엑셀 컬럼 매핑 (OpenAI)
- 상품명/SKU 기반 상품 자동 매칭
- 부피 기반 패킹 추천 계산
- 패킹 결과 엑셀 내보내기
- 대시보드 통계

없는 것:
- 인증/권한 관리
- 재고 차감
- 멀티 창고
- 실제 3D 좌표 기반 적재 최적화 (부피 기반만 존재)
